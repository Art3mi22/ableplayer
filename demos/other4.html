Able Player Fallback Test (using nested HTML content)

The examples on this page (one audio, one video) demonstrate Able Player's support for fallback content, nested within the <audio> or <video> tags. If a critical error is encountered that prevents the media from loading, the fallback content is presented in its place. Able Player displays the following fallback content, in descending order of precedence:

  1. Fallback HTML content nested within the <audio> or <video> tags
  2. An error message that explains why the media cannot be played. If a poster attribute is included on the <video> tag, that will be used to display an image in addition to the error message.
