Able Player for YouTube
Example 1, Captions Hosted on YouTube

This page features an example of Able Player being used to play a YouTube video. Its features include:

  • A described and non-described version of the video, both hosted on YouTube. Swap between the two versions using the Description button.
  • Captions and subtitles hosted on YouTube (NOTE: without locally-hosted captions, Able Player is unable to generate an interactive transcript).
  • Limited options in the Captions Preferences dialog (users can control the font size of captions)
