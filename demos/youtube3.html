Able Player for YouTube (Complex Example)

This page features a relatively complex example of Able Player being used to play a YouTube video. Its features include:

  • A described and non-described version of the video, both hosted on YouTube. Swap between the two versions using the Description button.
  • Captions, subtitles, and chapters, all hosted in local WebVTT files, and referenced within <track> elements.
  • An interactive transcript that combines all of the above text tracks.
  • External containers to which Able Player adds chapters and the interactive transcript.
