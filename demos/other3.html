Able Player Fallback Test (using JW Player)

The examples on this page (one audio, one video) automatically use JW Player rather than the usual Able Player Chrome. JW Player is not provided with Able Player, and must be licensed separately. If the <audio> or <video> tags include data-fallback="jw" and JW Player is properly installed, Able Player will fallback to JW Player if the user's browser doesn't support HTML5 media. The player can be forced to use JW Player by also adding the data-test-fallback attribute to the media element.
